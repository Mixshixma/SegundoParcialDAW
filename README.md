Carteles Anónimos

Sistema de tablón de anuncios efímero y anónimo desarrollado con arquitectura desacoplada.

    Frontend: HTML5, JavaScript (Fetch API) y Bootstrap 5 alojado en Netlify.

    Backend: API REST en PHP 8.2 con Docker, desplegada en Render.

    Base de Datos: MySQL gestionado en Aiven Cloud.

    Seguridad: Gestión de anuncios mediante Tokens únicos por publicación sin necesidad de registros.
